AEAC: Astrogod Endpoint Advanced Collection

AEAC comes with 2 tools:
  1. AstroShell a command line executable <= StdIn
  2. AstroServ an executable serving on port 8000 <= HTTP

Examples

astroserv  # Then open localhost:8000

astroshell astro_print_all_points "1,2:3,4.72"
astroshell astro_usage
astroshell astro_hi0
astroshell astro_hi1  Hello 
astroshell astro_hi2  Hello Nico
astroshell astro_msg  "Print a message"
